PuTTY 用户手册

PuTTY 是一个免费的 (MIT 协议发布的) Win32 Telnet 和 SSH 客户端. 这是 PuTTY 和它的 companion utilities PSCP, PSFTP, Plink, Pageant and PuTTYgen 的文档.

对于UNIX用户 this manual currently primarily documents the Windows versions of the PuTTY utilities. Some options are therefore mentioned that are absent from the Unix version; the Unix version has features not described here; and the pterm and command-line puttygen utilities are not described at all. The only Unix-specific documentation that currently exists is the man pages.
